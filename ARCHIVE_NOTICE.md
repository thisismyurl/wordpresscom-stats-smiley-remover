# Archived — 2026-05-02

This repository is read-only. The plugin is **still maintained** — just not here.

## Where the plugin lives now

The active distribution and support surface is the WordPress.org plugin directory. Install, update, and report issues there.

- **WordPress.org plugin page:** `https://wordpress.org/plugins/wordpresscom-stats-smiley-remover/`
- **Author page:** [profiles.wordpress.org/thisismyurl](https://profiles.wordpress.org/thisismyurl/)

## Why archive on GitHub?

This Is My URL has been consolidating the public surface — keeping development active on the modern plugin set ([thisismyurl-* repos](https://github.com/thisismyurl?tab=repositories)) and letting the smaller, focused legacy plugins be maintained from .org alone. The code shipped here is stable; if a real bug or compat issue surfaces, a fresh repo will be opened for it.
